Opciones Completion

  • Si se elige “Mensaje requerido”, entonces el blog se marcará como que se ha completado para un usuario que ha realizado el número especificado de envíos.

  • Si se elige “Comenario requerido”, el blog quedará marcado como completado para el usuario una vez que haya cumplimentado sus comenarios. The comments can be on anyone's post as long as it is within this blog.

Note that course and personal blogs are separate. Posts and comments to personal blogs don't count toward completion requirements for a course blog.
